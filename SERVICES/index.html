ΓΙΑΤΙ ΧΡΕΙΑΖΟΜΑΙ ΘΕΡΑΠΕΙΑ?

Η απόφαση για θεραπεία μπορεί να εγείρει πολλά ερωτήματα μέσα σας. "Χρειάζομαι πραγματικά θεραπεία; Θα με καταλάβει ο θεραπευτής μου;". Η ψυχοθεραπεία προσφέρει έναν ασφαλή και υποστηρικτικό χώρο για να εξερευνήσετε προκλήσεις, σχέσεις και επαναλαμβανόμενα μοτίβα που ίσως σας δυσκολεύουν. Eίτε αντιμετωπίζετε άγχος, κατάθλιψη, απώλεια, δυσκολίες στις σχέσεις ή απλώς νιώθετε ότι κάτι δεν πάει καλά, η θεραπεία μπορεί να σας βοηθήσει να κατανοήσετε καλύτερα τον εαυτό σας και να ανακαλύψετε νέους τρόπους διαχείρισης και εξέλιξης. Έαν νιώθετε οτι δεν μπορείτε να αντιμετωπίσετε αποτελεσματικά μια κατάσταση, ακόμη και αν έχετε καταβάλλει μεγάλη προσπάθεια, μπορείτε να ζητήσετε την γνώμη μου έγκαιρα, για την ταχύτερη και αποτελεσματικότερη αντιμετώπιση του αιτήματος σας.

ΤΙ ΝΑ ΠΕΡΙΜΕΝΩ

01

Θα ξεκινήσουμε με μια δωρεάν 30λεπτη τηλεφωνική συνομιλία για να έχω μια αίσθηση του είδους της υποστήριξης που αναζητάτε. Θα ζητήσω μερικές βασικές πληροφορίες για να σας γνωρίσω και έπειτα θα απαντήσω σε τυχόν ερωτήσεις που μπορεί να έχετε. Εάν αισθανθείτε ότι επιθυμείτε να συνεργαστούμε, θα προγραμματίσουμε την πρώτη διαδικτυακή συνεδρία.

02

Στην πρώτη συνεδρία, θα μάθω περισσότερα για εσάς και μαζί θα διερευνήσουμε τους στόχους και τις προσδοκίες που έχετε για τη θεραπεία. Στη συνέχεια θα εστιάσουμε μαζί στην λύση του αιτήματος σας, μέσω της καλύτερης κατανόησης του εαυτού σας και του τρόπου με τον οποίον σχετίζεστε με τους γύρω σας.

03

Η διάρκεια και ο αριθμός των συνεδριών θα εξαρτηθεί αποκλειστικά απο εσας και τους στόχους σας. Μόνο εσέις γνωρίζετε τις προσωπικές σας ανάγκες και τα όρια σας. Οποιαδήποτε στιγμη, μπορείτε να αναπροσαρμόσετε τη συχνότητα ανάλογα με την υποστήριξη που ψάχνετε. Πρωταρχικός μου στόχος είναι να επανακτήσετε την λειτουρικότητας σας απο τις πρώτες κιόλας συνεδρίες.

Ατομική Ψυχοθεραπεία

Βοηθάω ενήλικες, εφήβους και παιδιά να αντιμετωπίσουν τις δυσκολίες της ζωής, όπως είναι το άγχος, το τράυμα, οι σχεσιακές ανησυχίες και άλλα.

Συμβουλευτική και Ψυχοθεραπεία Ζεύγους

Προσφέρω σε εσάς και τον/την σύντροφο σας ένα πλάισιο για να χτίσετε μια πιο υγειή σχέση και να ευημερήσετε στη ζωή σας, ατιμετωπίζοντας τις καθημερινές δυσκολίες που ίσως προκαλέι η ρουτίνα.

Οικογενιακή Θεραπεία και Συμβουλευτική Γονέων

Ενισχύω την ικανότητα σας, ως γονέας, να επικοινωνήσετε αποτελεσματικά με το παιδί σας, με στόχο να εξαλείψετε τις συγκρούσεις και να αναθρεύσετε ενα ανεξάρτητο και καλά οργανωμένο άτομο.
